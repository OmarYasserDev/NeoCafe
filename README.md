# NeoCafe ☕

This is a new and responsive coffee shop website created with HTML and CSS.

## Features
- Responsive design.
- Smooth scrolling navigation.
- Interactive buttons and hover effects.
- Menu and gallery.
- Contact.
- Classic design.

## Skills Gained
From this project, I gained skills in:
- Responsive design.
- CSS Flexbox.
- Web design structure with HTML.
- Interactive styling of UI.

## The Purpose of the Project
The purpose of this project was to practice frontend development and build a modern cafe landing page.

## Tools
- HTML5
- CSS3
