NeoCafe
Hello and welcome to NeoCafe! Here you will find my creation - a beautiful responsive website for a coffee shop. This responsive website was created by me from scratch using HTML and CSS in order to design a simple yet elegant layout which a real cafe can use to promote its products.

What It Has
Here are some of the major things which were incorporated in the layout:

Fully Responsive Design: This makes it possible to view the layout on both small screen devices as well as desktop screens.

Smooth Navigation: Smooth scrolling is used in this layout to ease navigation.

Dedicated Pages: Complete menu design, gallery, and contact pages have been included in the layout.

Interactive UI: Several buttons and hover effects have been added to make the classic design look interactive.

Made with
The tools used for designing the layout are as follows:

HTML5

CSS3

Lessons Learned
The development of NeoCafe was an excellent exercise for improving my front-end skills. I had the chance to practically apply the knowledge of organizing a webpage using HTML and made extensive use of CSS Flexbox to implement a layout that works responsively and aligns properly. Moreover, I got the chance to play around with the interactive styles and different button states without importing any other library.
